** What changed

The root cause was an _off-by-one error_ in the date range query.
The old approach was +completely wrong+ and had been there since day one.

Use =deps.edn= to manage dependencies and ~lein repack~ to rebuild.

*** Implementation detail

The fix was straightforward:

#+begin_src clojure
(defn date-range [start end]
  (let [days (time/days-between start end)]
    (map #(time/plus start (time/days %))
         (range (inc days)))))
#+end_src
